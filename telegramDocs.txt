## Setup / Initialization

To connect your Mini App to the Telegram client, place this script in `<head>` **before any other scripts**:

```html
<script src="https://telegram.org/js/telegram-web-app.js?62"></script>
```

This makes `window.Telegram.WebApp` available.

## CloudStorage *(Bot API 6.9+)*

`Telegram.WebApp.CloudStorage` — key-value storage synced to Telegram cloud. Up to **1024 keys** per Mini App, key max **128 chars**, value max **4096 chars**. Keys: `A-Z`, `a-z`, `0-9`, `_`, `-`.

| Method | Description |
|---|---|
| `setItem(key, value[, callback])` | Stores value. `callback(error, stored)` |
| `getItem(key, callback)` | Retrieves value. `callback(error, value)` — value is empty string if not found |
| `getItems(keys, callback)` | Retrieves multiple values. `callback(error, values)` — object `{key: value}` |
| `removeItem(key[, callback])` | Removes key. `callback(error, removed)` |
| `removeItems(keys[, callback])` | Removes multiple keys. `callback(error, removed)` |
| `getKeys(callback)` | Gets all stored keys. `callback(error, keys)` — array of strings |

Example:
```javascript
const storage = Telegram.WebApp.CloudStorage;

// Save
storage.setItem("user_preference", "dark_mode", (err, stored) => {
  if (!err && stored) console.log("Saved!");
});

// Read
storage.getItem("user_preference", (err, value) => {
  if (!err) console.log("Value:", value);
});

// Get all keys
storage.getKeys((err, keys) => {
  console.log("All keys:", keys);
});
```

## BiometricManager *(Bot API 7.2+)*

`Telegram.WebApp.BiometricManager`

| Field | Type | Description |
|---|---|---|
| `isInited` | Boolean | Whether manager is initialized |
| `isBiometricAvailable` | Boolean | Whether biometric auth is available |
| `biometricType` | String | `"finger"`, `"face"`, or `"unknown"` |
| `isAccessRequested` | Boolean | Whether access has been requested |
| `isAccessGranted` | Boolean | Whether access was granted |
| `isBiometricTokenSaved` | Boolean | Whether a token is saved |
| `deviceId` | String | Unique device ID |

| Method | Description |
|---|---|
| `init([callback])` | Initialize manager. Must be called first. `callback()` when ready |
| `requestAccess(params[, callback])` | Request biometric permission. `params`: `{reason}`. `callback(granted)` |
| `authenticate(params[, callback])` | Authenticate user. `params`: `{reason}`. `callback(authenticated, token)` |
| `updateBiometricToken(token[, callback])` | Save/update token. `callback(updated)` |
| `openSettings()` | Open device biometric settings |

Example:
```javascript
const bio = Telegram.WebApp.BiometricManager;
bio.init(() => {
  if (bio.isBiometricAvailable) {
    bio.requestAccess({ reason: "Authenticate to access your account" }, (granted) => {
      if (granted) {
        bio.authenticate({ reason: "Confirm your identity" }, (ok, token) => {
          if (ok) console.log("Authenticated, token:", token);
        });
      }
    });
  }
});
```

## Accelerometer *(Bot API 8.0+)*

`Telegram.WebApp.Accelerometer`

| Field | Type | Description |
|---|---|---|
| `isStarted` | Boolean | Whether accelerometer is active |
| `x` | Float | X-axis acceleration (m/s²) |
| `y` | Float | Y-axis acceleration (m/s²) |
| `z` | Float | Z-axis acceleration (m/s²) |

| Method | Description |
|---|---|
| `start(params[, callback])` | Start tracking. `params`: `{refresh_rate}` in ms (min 20). `callback(started)` |
| `stop([callback])` | Stop tracking. `callback(stopped)` |

Example:
```javascript
const accel = Telegram.WebApp.Accelerometer;
accel.start({ refresh_rate: 100 }, (started) => {
  if (started) {
    Telegram.WebApp.onEvent("accelerometerChanged", () => {
      console.log(`X: ${accel.x}, Y: ${accel.y}, Z: ${accel.z}`);
    });
  }
});
```

## Passing Data to the Bot

### Method 1 — `sendData()` (Keyboard button Mini Apps only)
```javascript
// In your Mini App:
Telegram.WebApp.sendData(JSON.stringify({ action: "selected_date", date: "2024-12-25" }));
// This closes the Mini App and sends a service message to the bot.
```
Bot receives via `message.web_app_data.data`.

### Method 2 — `answerWebAppQuery()` (Inline button / attachment menu)
```javascript
// Mini App sends query_id to your server
fetch('/submit', {
  method: 'POST',
  body: JSON.stringify({ query_id: Telegram.WebApp.initDataUnsafe.query_id, data: "..." })
});
// Server calls Bot API: answerWebAppQuery(web_app_query_id, result)
```

## Validating Data Received via the Mini App

Always validate `initData` on the server. **Never trust `initDataUnsafe` directly.**

### Algorithm:
1. Parse the `initData` query string
2. Extract `hash` field
3. Sort remaining fields alphabetically
4. Create data-check string: `field=value\nfield=value\n...`
5. Compute `HMAC-SHA256(data_check_string, HMAC-SHA256("WebAppData", BOT_TOKEN))`
6. Compare with extracted `hash`

### Python example:
```python
import hashlib, hmac
from urllib.parse import parse_qsl

def validate_init_data(init_data: str, bot_token: str) -> bool:
    parsed = dict(parse_qsl(init_data))
    received_hash = parsed.pop("hash", None)
    
    data_check_string = "\n".join(
        f"{k}={v}" for k, v in sorted(parsed.items())
    )
    
    secret_key = hmac.new(b"WebAppData", bot_token.encode(), hashlib.sha256).digest()
    computed_hash = hmac.new(secret_key, data_check_string.encode(), hashlib.sha256).hexdigest()
    
    return hmac.compare_digest(computed_hash, received_hash)
```

### Node.js example:
```javascript
const crypto = require('crypto');

function validateInitData(initData, botToken) {
  const params = new URLSearchParams(initData);
  const hash = params.get('hash');
  params.delete('hash');
  
  const dataCheckString = Array.from(params.entries())
    .sort(([a], [b]) => a.localeCompare(b))
    .map(([k, v]) => `${k}=${v}`)
    .join('\n');
  
  const secretKey = crypto.createHmac('sha256', 'WebAppData')
    .update(botToken)
    .digest();
  
  const computedHash = crypto.createHmac('sha256', secretKey)
    .update(dataCheckString)
    .digest('hex');
  
  return crypto.timingSafeEqual(
    Buffer.from(computedHash, 'hex'),
    Buffer.from(hash, 'hex')
  );
}
```

### Validating Data for Third-Party Use *(Bot API 8.0+)*
Third-party services can validate without knowing the bot token:

```
secret_key = HMAC-SHA256(init_data.bot_id + ":" + init_data.domain, "WebAppDataThirdParty")
```

## Expanding / Viewport

```javascript
// Check if expanded
if (!Telegram.WebApp.isExpanded) {
  Telegram.WebApp.expand();
}

// Listen to viewport changes
Telegram.WebApp.onEvent("viewportChanged", ({ isStateStable }) => {
  if (isStateStable) {
    const height = Telegram.WebApp.viewportStableHeight;
    document.body.style.minHeight = height + "px";
  }
});
```

## Full-Screen Mode *(Bot API 8.0+)*

```javascript
// Request fullscreen
Telegram.WebApp.requestFullscreen();

// Exit fullscreen
Telegram.WebApp.exitFullscreen();

// Listen for fullscreen events
Telegram.WebApp.onEvent("fullscreenChanged", () => {
  console.log("Fullscreen:", Telegram.WebApp.isFullscreen);
});

Telegram.WebApp.onEvent("fullscreenFailed", ({ error }) => {
  console.error("Fullscreen failed:", error);
  // error: "UNSUPPORTED" | "ALREADY_FULLSCREEN"
});

// Lock orientation in fullscreen
Telegram.WebApp.lockOrientation();
Telegram.WebApp.unlockOrientation();
```

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Mini App</title>
  <script src="https://telegram.org/js/telegram-web-app.js?62"></script>

    body {
      background-color: var(--tg-theme-bg-color);
      color: var(--tg-theme-text-color);
      font-family: -apple-system, sans-serif;
      margin: 0;
      padding: 16px;
    }
    button {
      background-color: var(--tg-theme-button-color);
      color: var(--tg-theme-button-text-color);
      border: none;
      padding: 12px 24px;
      border-radius: 8px;
      cursor: pointer;
    }
  </style>
</head>
<body>
  <h1>Hello!</h1>
  <p id="user-name"></p>
  <button onclick="submitAction()">Submit</button>

  <script>
    const webapp = window.Telegram.WebApp;
    
    // Tell Telegram the app is ready (hides loading screen)
    webapp.ready();
    
    // Expand to full height
    webapp.expand();
    
    // Display user info
    const user = webapp.initDataUnsafe.user;
    if (user) {
      document.getElementById("user-name").textContent = 
        `Hello, ${user.first_name}!`;
    }
    
    // Setup main button
    webapp.MainButton.setText("Confirm");
    webapp.MainButton.show();
    webapp.MainButton.onClick(() => {
      webapp.MainButton.showProgress(false);
      // ... do work ...
      webapp.close();
    });
    
    function submitAction() {
      webapp.HapticFeedback.impactOccurred("medium");
      webapp.showConfirm("Are you sure?", (confirmed) => {
        if (confirmed) {
          webapp.sendData(JSON.stringify({ action: "confirmed" }));
        }
      });
    }
    
    // Theme change listener
    webapp.onEvent("themeChanged", () => {
      console.log("New scheme:", webapp.colorScheme);
    });
  </script>
</body>
</html>
```
